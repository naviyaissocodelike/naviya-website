# Images

Drop image files here and they'll be served at `/images/<filename>`.

Expected files (referenced from `src/components/VaultHome.jsx`):

- `about.jpg` — main polaroid in the About section
- `climbing.jpg` — Play tile
- `crossfit.jpg` — Play tile
- `hiking.jpg` — Play tile
- `nomadic.jpg` — Play tile

If a file doesn't exist, the site falls back to a styled placeholder with the label, so the layout always looks intentional.
